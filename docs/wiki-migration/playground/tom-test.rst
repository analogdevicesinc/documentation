Raspberry Pi Pico Tool for GMSL2 I2C Access
===========================================

On many occasions, it's helpful to check a few registers on a GMSL device for confirming settings, checking errors, or running a diagnostic check. The GMSL GUI Tool is an extremely useful tool to perform these tasks. But if there is not an EV kit available to run the GUI firmware, this tool provides command line I2C access and Python scripting capabilities for testing algorithms.
