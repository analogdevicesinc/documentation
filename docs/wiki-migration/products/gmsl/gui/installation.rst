Download and Installation of the GMSL GUI
=========================================

.. container:: group

   
   .. container:: half column

      The latest GMSL GUI can always be found on the bottom of the GMSL product pages, such as the :adi:`MAX96717 <en/products/max96717.html>`.

   
   .. container:: half column


   
   |image1|

--------------

.. container:: group

   
   .. container:: half column

      The GMSL GUI will also be updated to support the publicly available parts released on analog.com.

   
   .. container:: half column


   
   |image2|

--------------

.. container:: group

   
   .. container:: half column

      Extract the compressed folder and run the executable "GMSL SerDes Public GUI_Vx_x_x_x_Install.exe".

   
   .. container:: half column


   
   |image3|

--------------

.. container:: group

   
   .. container:: half column

      Also, install the Teensy uC driver for the microcontroller that is on the GMSL EV kits.

   
   .. container:: half column


   
   |image4|

--------------

.. container:: group

   
   .. container:: half column

      After installation of the GMSL GUI is complete, navigate to the folder and run the GUI from the Start Menu.

   
   .. container:: half column


   
   |image5|

--------------

.. container:: group

   
   .. container:: half column

      Now we can begin interfacing with the GMSL EV kits.

   
   .. container:: half column


   
   |image6|
