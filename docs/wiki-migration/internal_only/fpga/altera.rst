Tools
-----

ADI has a license to use the Altera provided `tools <http://www.altera.com/products/software/sfw-index.jsp>`__. While this does not currently include IP, this can be provided in the future. For more information, contact `Gordon Sterling <https://wiki.analog.com/gordon.sterling@analog.com>`__ or `Greg Duffy <https://wiki.analog.com/greg.duffy@analog.com>`__.

Boards
------

The `DN7002K10MEG <http://www.dinigroup.com/new/DN7002k10MEG.html>`__ is in use for pre-silicon hardware/software co-verification by the EEPG of the Processor-CPT. The board is made by the `Dini Group <http://www.dinigroup.com>`__. Contact `Joe Bastos <https://wiki.analog.com/joe.bastos@analog.com>`__ for more information.
