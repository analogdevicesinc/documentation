ADZS-ADuCM4050 User Guides
==========================

These user guides are designed to assist customers with the installation of CrossCore Embedded Studio and use of the ADZS-ADuCM4050LF EZ-KIT\ :sup:`®` or ADZS-ADuCM4050WL EZ-KIT\ :sup:`®` on Linux and Windows operating systems.

User Guides
-----------

-  :doc:`CrossCore Download & Installation Guide </wiki-migration/resources/eval/user-guides/adzs-aducm4050/crosscore_download_install_guide>`

::

     *Follow this guide to download and install CCES
   * CrossCore Quick Start User Guides for Linux and Windows
     * [[/resources/eval/user-guides/adzs-aducm4050/crosscore_quickstart_user_guide_for_linux|CrossCore Quick Start User Guide For Linux]]
       *Follow this guide to configure CCES on Linux to be used with the ADZS-ADuCM4050 EZ-KITs
   * [[/resources/eval/user-guides/adzs-aducm4050/crosscore_quickstart_user_guide|CrossCore Quick Start User Guide For Windows]]
       *Follow this guide to configure CCES on Windows to be used with the ADZS-ADuCM4050 EZ-KITs

For full documentation of the software please refer to the :adi:`CrossCore Embedded Studio <en/design-center/processors-and-dsp/evaluation-and-development-software/adswt-cces.html>` web site.

For full documentation of the hardware please refer to the :adi:`ADZS-ADuCM4050LF EZ-KIT <en/design-center/evaluation-hardware-and-software/evaluation-boards-kits/ADZS-U4050LF-EZKIT.html>` or :adi:`ADZS-ADuCM4050WL EZ-KIT <en/design-center/evaluation-hardware-and-software/evaluation-boards-kits/ADZS-U4050WL-EZKIT.html>` web sites.
