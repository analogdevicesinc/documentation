Hardware Details
================

This chapter contains hardware-related information about the available ADAS1021 evaluation boards. Each sub-section contains a general description of the board, detailed descriptions of the connectors, jumpers, and buttons. It also provides links to the Schematics, Bill of materials, design projects, and Technical documentation. It also gives internal links to the provided example software projects

KEY FEATURES:

-  5 or 10 ECG electrode paths capable of demonstrating 1-lead to 12-leads of ECG data
-  AC and DC lead off detection
-  Respiration Measurement on RA, LA and LL
-  Independent Pace measurement on all ECG channels
-  Calibration features
-  Common RLD circuit
-  ESIS Filtering on all ECG signal paths
