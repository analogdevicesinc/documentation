1. Introduction
===============

The :adi:`EVAL-ADAS1021ARDZ` is a 12-lead ECG, respiration and Pace detect development board. The board contains 2x ADAS1021 ECG AFE’s in a master/slave configuration. The AFE’s are powered by the :adi:`ADP5320` power management unit which generates all necessary supplies for the board from a single 5V input which is taken from the Arduino header.

The primary function of the EVAL-ADAS1021ARDZ evaluation board is to demonstrate the ADAS1021 integrated ECG device for medical instrumentation. A full description of the ADAS1021 is available in the data sheet and should be consulted when utilizing this user guide. This evaluation board is used to evaluate the ADAS1021 and its features. The EVAL-ADAS1021ARDZ is compatible with Arduino R3 devices. C/C++ drivers for the use of the EVAL-ADAS1021ARDZ are available with examples available for the Nordic NRF52840-DK.

This guide is structured as follows:

-  Hardware Details - Contains hardware-related information about the base board
-  Software Details - Contains software-related information about the base board
