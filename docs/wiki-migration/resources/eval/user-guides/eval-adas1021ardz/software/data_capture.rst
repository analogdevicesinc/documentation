Data Capture and Transfer
=========================

-  For data capture and transfer, click on the ‘Capture’ tab screen to setup data capture and transfer.
-  Click on the ‘\ **…**\ ’ button (red circle in the figure below) to specify the capture files name and location on your local machine. Then create and save file.
-  Set Delimiter mode to ‘\ **comma**\ ’ (blue circle in the figure below) .
-  Click on “\ **Start Overwrite**\ ” button to begin capture.
-  Click on “\ **Stop Capture**\ ” button to end capture.
-  Use a simple text editing program such as Microsoft Notepad to open and retrieve your data from the capture file.
-  Use the “\ **Select All**\ ” and “\ **Copy**\ ” features of the Edit function in Notepad to copy and paste CSV text data into Microsoft Excel.
-  Use the “Text to Columns” feature of the Data function in Excel to break the CSV text data into separate columns.
-  Analyse and graph the data as desired.
