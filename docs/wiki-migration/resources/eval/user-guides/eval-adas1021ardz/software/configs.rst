Device Configurations
=====================

The EVAL-ADAS1021ARDZ has various configurations that a user can alter for their own use cases, such as ECG, Respiration, FIFO setup etc. These primary configurations can be altered and adjusted by interfacing with certain structures defined in the evaluation software provided. Specifically, these structures are defined in the ADAS1021Lib.h file and it is recommended to use the predefined masks provided in the ADAS1021.h file to read/write data from/to the configuration registers on the part. Important configuration structures are outlined in the subsequent tables, followed by the resultant of changing individual structure members to the overall configuration. <fc #ff0000>Note text in bold red refers to a predefined mask in the provided ADI software package.</fc>

**Structure Name: ECG_RespFifoConfig_Type** : Adjustments to this structure’s members allows the user to directly interact with the ECG/Respiration FIFO register address. This register address is used to access the ECG/Respiration FIFO. Each read of this register reads one data word from the head of the FIFO. Please consult the ADAS1021 Datasheet for more information.

+--------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Member Name  | Function                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
+==============+================================================================================================================================================================================================================================================================================================================================================================================================================================================================================================================================================================+
| DeltaReset   | Restart delta calculation logic on the application of a configuration change. **When 0**/ <fc #ff0000>bFALSE</fc>: the delta calculation logic is not reset when a configuration change is applied. **When 1**/ <fc #ff0000>bTRUE</fc>: the delta calculation logic is reset during a configuration change.                                                                                                                                                                                                                                                    |
+--------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| ChannelSel   | Enable loading of Respiration magnitude data into the FIFO. User may choose a combination of the following channels: <fc #ff0000>FIFOECG_CH1</fc>, <fc #ff0000>FIFOECG_CH2</fc>, <fc #ff0000>FIFOECG_CH3</fc>, <fc #ff0000>FIFOECG_CH4</fc>, <fc #ff0000>FIFOECG_CH5</fc>.                                                                                                                                                                                                                                                                                     |
+--------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| FifoBitWidth | Select 19-bit or 20-bit mode for delta calculations. **When 0**/ <fc #ff0000>FIFOBITW_19B</fc>: the FIFO loader state machine takes 19-bits of data from the ECG channels, and use these values for the delta calculations, or pack 19-bit data into the FIFO, if ECG_RESP_FIFO_ECG_ABS_LD_EN bit is set. **When 1**/ <fc #ff0000>FIFOBITW_20B</fc>: the FIFO loader state machine takes 20-bits of data from the ECG channels, and use these values for the delta calculations, or pack 20-bit data into the FIFO, if ECG_RESP_FIFO_ECG_ABS_LD_EN bit is set. |
+--------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| FifoMode     | Enable loading of ECG Absolute Channel data into FIFO. **When 0**/ <fc #ff0000>FIFOMODE_DELTA</fc>: delta mode from the enabled ECG channels is loaded into the shared ECG/Respiration FIFO. **When 1**/ <fc #ff0000>FIFOMODE_ABS</fc>: absolute data from the enabled ECG channels is loaded into the shared ECG/Respiration FIFO.                                                                                                                                                                                                                            |
+--------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

**Structure Name: RespConfig_Type** : The respiration system is capable of handling changes to its configuration; however, there is a sequence of events associated with such changes. When a configuration change is required, the user first programs the RESP_CONTROL register with the updated configuration. This structure allows the user to directly choose the respiration configuration by adjusting the below structure members. Please consult the ADAS1021 Datasheet for more information.

+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Member Name      | Function                                                                                                                                                                                                                                                                                                                           |
+==================+====================================================================================================================================================================================================================================================================================================================================+
| RespElectrodeSel | Select the pair of electrodes to use for respiration measurement. A configuration update request is not required for changes to this bit field. **001**/ <fc #ff0000>RESP_LA_RA</fc>: LA_RA. **010**/ <fc #ff0000>RESP_LA_RL</fc>: LA_RL. **011**/ <fc #ff0000>RESP_LL_RA</fc>: LL_RA. **100**/ <fc #ff0000>RESP_LL_RL</fc>: LL_RL |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| RespMagOnly      | Allow respiration magnitude only to FIFO. **When 0**/ <fc #ff0000>bFALSE</fc>: the respiration channel outputs magnitude and phase samples to be written to the FIFO. **When 1**/ <fc #ff0000>bTRUE</fc>: the respiration channel outputs magnitude only to be written to the FIFO.                                                |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| RespRate         | Respiration output data rate select. Use this bit to control the output data rate of the respiration channel. **When 0**/ <fc #ff0000>RESPRATE_100SPS</fc>: output data rate is 100 SPS. **When 1**/ <fc #ff0000>RESPRATE_25SPS</fc>: output data rate is 25 SPS.                                                                  |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| RespPgaGn        | Select PGA gain of respiration measurement. Configuration update request procedure applies to this bit. **00**/ <fc #ff0000>RESPPGA_24</fc>: Gain = 24. **01**/ <fc #ff0000>RESPPGA_48</fc>: Gain = 48. **10**/ <fc #ff0000>RESPPGA_1</fc>: Gain = 1 (bypass mode) **11**: Reserved                                                |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

**Structure Name: ECGConfig_Type** : The ECG control registers allow the user to arrange the input stage to the ECG channels in a variety of different ways. This structure allows the user to directly choose the ECG configuration by adjusting the below structure members. Please consult the ADAS1021 Datasheet for more information.

+-------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Member Name | Function                                                                                                                                                                                                                                                                                                                                                             |
+=============+======================================================================================================================================================================================================================================================================================================================================================================+
| StartEcg    | Starts the ECG digital signal processing. Setting this bit (<fc #ff0000>bTRUE</fc>) initiates and aligns the clocking in ECG, respiration, DCLO, timestamp, ACLQ, calibration, FIFO subsystem and Pace.                                                                                                                                                              |
+-------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| ECG_1LA1P   | ECG channel mode switch. **When 0**/<fc #ff0000>OPEN</fc>: the switch is open. **When 1**/<fc #ff0000>CLOSED</fc>: this bit enables ECG_1LA1P switch.                                                                                                                                                                                                                |
+-------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| ECG_3LL2P   | ECG channel mode switch. **When 0**/<fc #ff0000>OPEN</fc>: the switch is open. **When 1**/<fc #ff0000>CLOSED</fc>: this bit enables ECG_3LL2P switch.                                                                                                                                                                                                                |
+-------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| ECG_3LL3P   | ECG channel mode switch. **When 0**/<fc #ff0000>OPEN</fc>: the switch is open. **When 1**/<fc #ff0000>CLOSED</fc>: this bit enables ECG_3LL3P switch.                                                                                                                                                                                                                |
+-------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| ECG_4V14P   | ECG channel mode switch. **When 0**/<fc #ff0000>OPEN</fc>: the switch is open.\ **When 1**/<fc #ff0000>CLOSED</fc>: this bit enables ECG_4V14P switch.                                                                                                                                                                                                               |
+-------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| ECG_5V25P   | ECG channel mode switch. **When 0**/<fc #ff0000>OPEN</fc>: the switch is open. **When 1**/<fc #ff0000>CLOSED</fc>: this bit enables ECG_5V25P switch.                                                                                                                                                                                                                |
+-------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| EcgChanEn   | Individual ECG channel enable bits. **When set 1**: these bits enable the relevant ECG channel. **When clear 0**: the corresponding ECG channel is disabled. 00001: <fc #ff0000>ECG_CH1</fc> enable. 00010: <fc #ff0000>ECG_CH2</fc> enable. 00100: <fc #ff0000>ECG_CH3</fc> enable. 01000: <fc #ff0000>ECG_CH4</fc> enable. 10000: <fc #ff0000>ECG_CH5</fc> enable. |
+-------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| EcgOdr      | ECG channel output date rate selection for all channels.\ **When 1**/ <fc #ff0000>ECGODR_300HZ</fc>: ECG output data rate is 300Hz. **When 0**/ <fc #ff0000>ECGODR_600HZ</fc>: ECG output data rate is 600Hz                                                                                                                                                         |
+-------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| EcgFilter   | Frequency selection for mains power line filtering **When 1**/ <fc #ff0000>ECG_50HZ</fc>: power line harmonics filtering is set for 50Hz. **When 0**/ <fc #ff0000>ECG_60HZ</fc>: power line harmonics filtering is set for 60Hz.                                                                                                                                     |
+-------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
