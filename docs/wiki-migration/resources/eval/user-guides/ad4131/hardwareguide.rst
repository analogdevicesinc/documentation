Hardware Guide
==============

This section of the user guide will go through the hardware on the EV-AD4131WASDZ-U1 board and will provide an explanation of the AD4131 itself, the hardware link options available on the board, how to power the part etc.

====Device Description==== The AD4131 is a low power, low noise, complete analog front end for high precision measurement applications. It contains a low noise, 16-bit Σ-Δ ADC. It can be configured to have eight differential inputs or 15 single-ended or pseudo-differential inputs. The on-chip low noise instrumentation amplifier means that signals of small amplitude can interface directly to the ADC. Other on-chip features include a low drift 1.25/2.5 V reference, excitation currents, reference buffers, and multiple filter options. Complete specifications for the AD4131 are provided in the product data sheet and must be consulted in conjunction with this user guide when using the evaluation board. Full details about the EVAL-SDP-CB1Z/ EVAL-SDP-CK1Z are available at analog.com.

| ====Set-up Procedures==== After following the instructions in the Software Installation Procedures section, set up the evaluation and SDP boards as detailed in this section. **Warning**
| The evaluation software and drivers must be installed before connecting the EV-AD4131WASDZ-U1 evaluation board and EVAL-SDP-CB1Z/ EVAL-SDP-CK1Z board to the USB port of the PC to ensure the PC correctly recognizes the evaluation system.
| **Configuring the Evaluation and SDP Boards**

-  Connect the SDP board to Connector A or Connector B on the EV-AD4131WASDZ-U1 board. Screw the two boards together using the plastic screw and washer set included in the evaluation board kit to connect the boards firmly together.
-  Connect the SDP board to the PC using the supplied USB cable.

| ====Power Supplies==== The evaluation board receives power through the EVAL-SDP-CB1Z/ EVAL-SDP-CK1Z when connect to the PC via USB . Linear regulators generate the required power supply levels from the applied VIN rail. The regulators used are the ADP150ACBZ-3.3 (U7) and the ADP150ACBZ-1.8 (U4), which supply 3.3 V and 1.8 V, respectively, to AVDD of the ADC. The 3.3 V ADP150ACBZ-3.3(U10) delivers 3.3 V to the IOVDD pin of the AD4131. When a split power supply is used, apply the AVSS voltage to the -1.8V ADP7182AUJZ-1.8 (U14) to the AVss pin of the AD4131. Each supply is decoupled at the point where it enters the board and again at the point where it connects to each device.
| ====Serial Interface==== The EV-AD4131-8ASDZ-U1/EV-AD4131WASDZ-U1 evaluation board connects via the serial peripheral interface (SPI) to the Blackfin® ADSP-BF527 on the EVAL-SDP-CB1Z. There are four primary signals: CS, SCLK, DIN, and DOUT/nRDY (all are inputs, except for DOUT/nRDY, which is an output). To operate the EV-AD4131-8ASDZ-U1/EV-AD4131WASDZ-U1 in standalone mode, the AD4131-8 serial interface lines can be disconnected from the 120-pin header by removing the 0 Ω links, R9 through R13. The test points can then be used to fly-wire the signals to an alternative digital capture setup.

| ====Reference Options==== The EV-AD4131WASDZ-U1 includes an external 2.5 V reference (the ADR391) and an internal 1.25 V or 2.5 V reference. The default operation uses the external reference input, which accepts the 2.5 V ADR391 on the evaluation board. Select the reference used for a conversion by choosing the reference in the configuration registers associated with Config 0 to Config 7. Switch between using the internal reference and external reference by accessing the AD4131-8 registers.
