AD4190-4 Customer Evaluation Board User Guide
=============================================

The EVAL-AD4190-4ARDZ evaluation kit features the AD4190-4, a multiplexed sigma delta analog to digital converter (ADC) 10nV at 3.8SPS(Gain = 128) ultra low noise core, integrated PGA and programmable digital filter offering output data rates from 62.5ksps to 3.8sps.

The PC software fully configures the AD4190-4 device register functionality and provides DC analysis , histograms, and associated noise analysis for ADC performance evaluation.

The AD4190-4 Family
-------------------

The AD4190-4 family of ADCs offers the highest integration and feature set required in the design of a multi-sensor measurement and is truly a platform measurement device, dc performance that enables instrumentation and industrial system designers to design across multiple measurement requirements for both isolated and non-isolated applications.

All devices in the family offer excellent DC performance. Here the integrated PGA can easily be bypassed if ultra low distortion measurement is required. The ultra low noise integrated PGA supports gains from X0,5 to X128, enabling an input span of +/- 2 \* VREF/gain – See the PGA section for more details. The AD4190-4 devices include a rich feature set from integrated low drift 5ppm/C reference, and key building blocks required for sensor excitation current and sensor biasing. The family of parts offer an extremely robust and trusted measurement solution through the integration of a large number of of measurement diagnostics

Quick Start Guide
=================

The following steps highlight the process to begin using the evaluation board.

Equipment Required
------------------

-  :adi:`EVAL-AD4190-4ARDZ evaluation board <en/products/ad4190.html>`
-  PC running Windows with a USB2.0 port and software installed.
-  Controller Board

   -  Option A: :adi:`EVAL-SDP-CK1Z <en/design-center/evaluation-hardware-and-software/evaluation-boards-kits/sdp-k1.html>` and a USB-C cable

Getting started
---------------

.. warning::

   Ensure the SDP board is not connected to the USB port of the PC


-  Install the :adi:`Ace Plugin <en/design-center/evaluation-hardware-and-software/evaluation-development-platforms/ace-software.html?doc=EVAL-AD7383FMCZ-ug-1770.pdf>`

   -  :doc:`Ace Plugin Install guide available here </wiki-migration/resources/eval/user-guides/ad4170/softwareprocedures>`

-  Connect the EVAL-AD4190-4ARDZ to the controller board

   -  **Option A:** Connect the EVAL-AD4190-4ARDZ to the EVAL-SDP-CK1Z

      -  Using the Arduino Connectors
