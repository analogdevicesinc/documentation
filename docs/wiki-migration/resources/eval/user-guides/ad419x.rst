**\*AD419x**\ \* HDL Reference Design
=====================================

Overview
--------

**\*Some specifications about the board, the chip, etc. Typically the information found on the** :adi:`en/products` **website**\ \*

The AD4190-4 is a low noise, completely integrated analog front end for high precision measurement applications. The device contains a low noise, 24-bit Σ-Δ analog-to-digital converter (ADC), and can be configured to have 4 differential inputs or 8 single-ended or pseudo differential inputs. The on-chip low noise stage ensures that signals of small amplitude can be interfaced directly to the ADC. The AD4190-4 establishes the highest degree of signal chain integration. The device contains a precision, low noise, low drift internal band gap reference and accepts an external differential reference, which can be internally buffered. Other key integrated features include programmable low drift excitation current sources, burnout currents, and a bias voltage generator, which sets the common-mode voltage of a channel to AVDD/2. The low-side power switch enables the user to power down bridge sensors between conversions, ensuring the absolute minimal power consumption of the system. The device also allows the user the option of operating with either an internal clock or an external clock.

Building the HDL project
------------------------

ADI does not distribute the bit/elf files of these projects so they must be built from the sources available :git-hdl>`__. To get the source you must `clone <https::`here </git-scm.com/book/en/v2/Git-Basics-Getting-a-Git-Repository>` the HDL repository.

Then go to the **\*PROJECT LOCATION WITHIN HDL (EX: projects/ad419x/coraz7s or projects/ad419x/de10nano)**\ \* location and run the make command by typing in your command prompt:

**Linux/Cygwin**

::

   user@analog:~$ cd hdl/projects/ad419x/coraz7s
   user@analog:~/hdl/projects/ad419x/coraz7s$ make

| Check :doc:`this guide </wiki-migration/resources/tools-software/linux-software/kuiper-linux>` on how to prepare your SD card with the proper boot files.
| A more comprehensive build guide can be found in the :doc:`HDL User Guide </wiki-migration/resources/fpga/docs/hdl>`.
