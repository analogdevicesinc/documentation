AD4131 User Guide
=================

| 
| The EV-AD4131WASDZ-U1 evaluation kit features the AD4131-8 16-bit, ultra-low power, low noise analog-to-digital converter (ADC).
| The EV-AD4131WASDZ-U1 board connects to the USB port of the PC by connecting to the EVAL-SDP-CB1Z/ EVAL-SDP-CK1Z motherboard. A 5V USB supply via the PC is regulated to 3.3 V to supply the AD4131 and support all necessary components.
| The AD4130 ACE Plugin fully configures the AD4131 device register functionality and provides dc time domain analysis in the form of waveform graphs, histograms, and associated noise analysis for ADC performance evaluation.
| The EV-AD4131WASDZ-U1 is an evaluation board that allows the user to evaluate the features of the ADC. The user PC software executable controls the AD4131 over the USB through the EVAL-SDP-CB1Z/ EVAL-SDP-CK1Z System Demonstration Platform (SDP) board.
| Full specifications on the AD4131 are available in the product data sheet, which should be consulted in conjunction with this user guide when working with the evaluation board.
| |image1|
| ====Features====

-  Full featured evaluation board for the AD4131
   \* PC control in conjunction with Analog Devices, Inc. System
   \* Demonstration Platform (EVAL-SDP-CB1Z/ EVAL-SDP-CK1Z)
   \* PC software for control and data analysis (time domain)
   \* Standalone capability
   ====Documents Needed====
-  AD4131 data sheet
   \* EV-AD4131WASDZ-U1 user guide
   ====Required Software====
-  AD4131 ACE Plugin
   ====Equipment Needed====
-  EV-AD4131WASDZ-U1 evaluation board
   \* EVAL-SDP-CB1Z/ EVAL-SDP-CK1Z System Demonstration Platform
   \* DC signal source
   \* USB cable
   \* PC running Windows with USB 2.0 port
   =====Quick Start Guide=====

| To begin using the evaluation board, do the following:
| \* With the EVAL-SDP-CB1Z/ EVAL-SDP-CK1Z board disconnected from the USB port of the PC, install the ACE software (can be downloaded online). Restart the PC after the software installation is complete. (For complete software installation instructions, see the Evaluation Software section.)

-  Connect the EVAL-SDP-CB1Z board to the EV-AD4131-8ASDZ-U1/EV-AD4131WASDZ-U1 board.

::

           *{{:resources:wechat_image_20210401114013.jpg?300|}}   {{:resources:wechat_image_20210401114007.jpg?300|}}
   * Screw the two boards together using the plastic screw and washer set included in the evaluation board kit to connect the boards firmly together.
           *{{:resources:wechat_image_20210401114000.jpg?300|}}
   * Connect the SDP board to the PC using the supplied USB cable. If you are using Windows® XP, you may need to search for the EVAL-SDP-CB1Z/ EVAL-SDP-CK1Z drivers. Choose to automatically search for the drivers for the SDP board if prompted by the operating system.\\
           *{{:resources:wechat_image_20210401123147.jpg?300|}}
   * From the Programs menu, go to the Analog Devices subfolder, and click ACE to launch the AD4130 ACE Plugin (see the Launching the Software section).
           *{{:resources:screenshot_2021-04-01_123406.png?400|}}
