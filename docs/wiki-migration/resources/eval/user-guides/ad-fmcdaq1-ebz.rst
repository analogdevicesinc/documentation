AD-FMCDAQ1-EBZ User Guide
=========================

The :adi:`AD-FMCDAQ1-EBZ <en/design-center/evaluation-hardware-and-software/evaluation-boards-kits/eval-ad-fmcdaq1-ebz.html#eb-overview>` is an FMC board for the high speed DAC :adi:`AD9122` & ADC :adi:`AD9684`. While the complete chip level design package can be found on the ADI product pages of these converters, information on the card, and how to use it, the design package that surrounds it, and the software which can make it work, can be found here.

The purpose of the :adi:`AD-FMCDAQ1-EBZ <en/design-center/evaluation-hardware-and-software/evaluation-boards-kits/eval-ad-fmcdaq1-ebz.html#eb-overview>` is a data acquisition platform that connects the analog world using FMC to the FPGA.
