ADAR3003,3002 (Longs Peak) ANALOG BEAMFORMER
============================================

(UNDER CONSTRUCTION)
====================

GENERAL DESCRIPTION
===================

The Ka Band Development Kit is an analog beamforming designed for testing the performance of the ADAR3002. The ADAR3002 is an 17 GHz to 21 GHz, 8-Channel, Dual Beam, Ka Band Beamformer IC. This Beamformer has 2 Channels per antenna element enabling dual polarization. This kit also contains a backplane that downconverts and powers the analog power beamformers.

The Ka Band Development Kit consists of 256 RF Channels, 64 RF cells, and 4 RF Tiles each cell contains a single ADAR3002, each tile contains 16 ADAR3002s. RF Outputs on the Tile are brought out to a WSMP connector to the front of the backplane. There is a 12V power input and all required voltage rails are generated by the backplane. There is a required 50 MHz Reference Frequency for the downconverter. Digital control of the beamformers and backplane is enabled using a dual PMOD interface. Control logic signals for the board are expected to be 3.3 logic with on board level translation to 1.8 logic.

BOARD DESIGN
============

The ADAR3002 Ka Band Dev Kit evaluation boarrd has integrated antennas and 2 WSMP RF output connectors for each of the 4 respective tiles. There are 64 antennas per tile, making up 256 antennas total.

A 6-pin Molex "Mini-Fit Jr." connectors are provided to apply the required 12V power supply.

Power Supply
------------

The ADAR1000-EVAL1Z board must be powered from the included power supply with a voltage level of 12V. There is an on-board power management tree which generates the required voltage rails for all of the associated parts.
