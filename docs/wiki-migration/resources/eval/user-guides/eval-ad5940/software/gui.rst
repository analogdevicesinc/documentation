GUI SETUP
=========

• Download the nRF Connect app from the Nordic website: https://www.nordicsemi.com/Software-and-tools/Development-Tools/nRF-Connect-for-desktop and install the software.

• Open the ADAS1021Eval_Setup folder, provided in the evaluation software suite, and double click on the exe file to run the GUI installer.


|image1|

• Follow the on-screen instructions to install the GUI.

• Next, open the nRF Connect app and open the Programmer interface.


|image2|

• Ensure the Nordic board is connected to the PC and select it under the Select Device tab as illustrated below.


|image3|

• Drag and drop the adas1021-NRF52840.hex file onto the nRF Connect main screen, or alternatively use the ‘Add HEX File’ button, and then click on Write on the right-hand pane (Note: The Write button may appear greyed out but will still function when clicked).


|image4|

• Close nRF Connect and power cycle the Nordic board

• Open the GUI tool and select the correct COM port |image5| • Subsequently, click on the HW Reset button. |image6| • Under the Device Configuration Setup download the Default_SingleMode setup file.

• Navigate to the ECG tab to modify settings as required.

• Click Measure to begin measurements.
