Firmware Examples
=================

The EVAL-ADAS1021ARDZ is supported with a full SDK (Software development Kit) that features a rich set of block level and application level example projects for the ADAS1021 and ADP5320 devices. Example projects for ECG and respiration measurements are available in the examples folders which have been compiled for the NRF52840 microprocessor. The projects have been compiled in the ARM IAR compiler version 8.32.3 but can be easily imported to newer revision of the software.
