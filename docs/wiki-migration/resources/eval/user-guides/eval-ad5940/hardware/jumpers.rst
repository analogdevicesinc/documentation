JUMPERS AND RESISTOR OPTIONS
============================

There are a number of jumpers included on this board for flexibility and ease of configuration. On receipt of the board, the jumpers are in the default state as described in the following table :-

+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| Jumper | Description                                     | Condition                                                                                                |
+========+=================================================+==========================================================================================================+
| SELBAT | Input selector to ADP5320 switch mode converter | Option A: 3V Shield., Option B: 5V0 Shield Default                                                       |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| AVDD5V | ADAS1021 AVDD5V power supply source selector    | Default Inserted, Remove for external power supply                                                       |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| 5VSINK | ADAS1021 5VSINK optional input                  | Default Removed                                                                                          |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| IV8    | ADAS1021 1V8 power supply source selector       | Default Inserted, Remove for external power supply                                                       |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| DVDD   | ADAS1021 1.2V DVDD power supply source selector | Default Inserted, Remove for external power supply                                                       |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| IOVDD  | ADAS1021 IOVDD power supply source selector     | Default Inserted, Remove for external power supply                                                       |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| JP4    | U3 I2C memory Write protect                     | Default inserted. Write enabled. Remove to enable write protect                                          |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| JP5    | ADP5320 address selector                        | Option A: High ADPD=0x28, Option B:Low=0x3A(DEFAULT), Option C: 400k to GND 0x38 removing JP gives float |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| JP6    | ADP5320 Enable/PG                               | Option A: PMU_EN Arduino header GPIO enable. Option B: VDD enable. Option C: GND disable                 |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| JP8    | I2C memory address selector                     | Option A: default address: Write CMD: 0xA2 Read CMD: 0xA3                                                |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| P6     | Connects ADP5320 VAUX Pin to 5V0_SHIELD supply  | Default open                                                                                             |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| P9     | ADP5320 PIN5 selector                           | Default position B, 3V3 Shield. Position A selects P10 Output                                            |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
| P10    | Option A input to P9 PVIN5 selector             | Position A VOUT1, Position B VOUT2                                                                       |
+--------+-------------------------------------------------+----------------------------------------------------------------------------------------------------------+
