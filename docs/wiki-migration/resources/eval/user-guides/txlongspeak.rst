ADAR3003,3002 (Longs Peak) ANALOG BEAMFORMER
============================================

(UNDER CONSTRUCTION)
====================

GENERAL DESCRIPTION
===================

The Tx Ka Band Development Kit is an analog beamforming designed for testing the performance of the ADAR3003. The ADAR3003 is an 27 GHz to 31 GHz, 8-Channel, Ka Band Beamformer IC. This Beamformer has 2 Channels per antenna element enabling dual polarization. This kit also contains a backplane that upconverts and powers the analog beamformers.

The Ka Band Development Kit consists of 256 RF Channels, 64 RF cells, and 4 RF Tiles each cell contains a single ADAR3003, each tile contains 16 ADAR3003s. RF Outputs on the Tile are brought out to a WSMP connector to the front of the backplane. There is a 12V power input and all required voltage rails are generated by the backplane. There is a required 50 MHz Reference Frequency for the upconverter. Digital control of the beamformers and backplane is enabled using a shrunken PMOD interface. Control logic signals for the board are expected to be 3.3 logic with on board level translation to 1.8 logic.

BOARD DESIGN
============

Power Supply
------------

The ADAR3003 Ka Band Dev Kit evaluation board has integrated antennas and 1 WSMP RF output connectors for each of the 4 respective tiles. There are 64 antennas per tile, making up 256 antennas total.

A 6-pin Molex "Mini-Fit Jr." connectors are provided to apply the required 12V power supply.
