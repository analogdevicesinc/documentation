AD-FMCVNA8-EBZ: 8-Port Vector Network Analyzer Card
===================================================

The AD-FMCVNA8-EBZ is a plug-in card for the Xilinx ZCU102 that realizes a full 8-port vector network analyzer (VNA) reference design. It performs S-parameter measurements for devices with up to 8 RF ports, and frequencies up to 20GHz. Software provided with the board includes Short-Open-Load-Thru (SOLT)-based

Hardware Configuration
----------------------

-  The AD-FMCVNA8-EBZ (rev B) plugs into the ZCU102 HPC0 FMC socket (closest to power switch)
-  The ZCU102 and VNA plug-in card each have their own power adapter (both are needed).
-  A PC (laptop) is needed to control and display the output of the ZCU.
-  Mini-USB (J83) on the ZCU (second socket from the edge) should be connected to the PC (included in the ZCU kit)
-  An ethernet cable should be connected between the PC and the ZCU; PC should setup a local network to the ZCU.
-  A >=16Gb SD card with bootable Linux image (for AD-FMCVNA8-EBZ revB), inserted into the ZCU SD card socket.
