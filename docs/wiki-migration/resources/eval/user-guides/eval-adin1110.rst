EVAL-ADIN1100FMCZ User Guide
============================

The :adi:`ADIN1100` is a low power single port 10BASE-T1L transceiver designed for industrial Ethernet applications and is compliant with the IEEE 802.3cg Ethernet standard for long reach 10 Mb/s Single Pair Ethernet. The EVAL-ADIN1100FMCZ allows simplified evaluation of the key features of the ADIN1100 Ethernet physical layer (PHY).
