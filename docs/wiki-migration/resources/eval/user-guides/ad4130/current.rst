Current Consumption Test
========================

Introduction
------------

The current consumption test is to carried out by measuring the voltage difference between AVDD and IOVDD. With two stuffed resistor on the EVAL-AD41300-8WASDZ (R96, R97) connecting to the IOVDD and AVDD in series and use ohms law to calculate the current consumption of the ADC.

Current Consumption Test Configuration
--------------------------------------

To gather test sample results and comparing the performance with different duty cycle modes, PGA gains from the ADC.

Jumper position
~~~~~~~~~~~~~~~

-  LK8, LK9 to be connected.
-  LK10, LK11 to be connected.
-  T_AVDD, T_IOVDD to be connected at position B.

Measuring Procedure
-------------------

-  Measure the accurate resistance of R96 and R97 using a multimeter.
-  With the register settings above, when AINP and AINM set to AIN2 and AIN3, we run the sample test and obtain the mean value of the voltage difference of IOVDD, we then use ohms law and calculate the IOVDD current with R96.
-  When AINP and AINM set to AIN6 and AIN7, we run the sample test and obtain the mean value of the voltage difference of AVDD, we then use ohms law and calculate the AVDD current with R97.
