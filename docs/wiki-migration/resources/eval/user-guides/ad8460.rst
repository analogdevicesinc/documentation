**\*AD8460**\ \* HDL Reference Design
=====================================

Overview
--------

**\*Some specifications about the board, the chip, etc. Typically the information found on the** :adi:`en/products` **website**\ \*

The AD8460 is a high power, high speed arbitrary waveform driver optimized for large dynamic currents into capacitive loads. It is comprised of a 14-bit high speed DAC, a high-voltage high-current analog driver, and fault monitoring and protection. The AD8460 is ideally suited for demanding applications such as arbitrary waveform generation, programmable power supplies, and LCD/OLED panel formation.

A proprietary high-voltage BCDMOS process and innovative architecture from Analog Devices, Inc., enable this high-performance driver to deliver 1A continuous into a resistive or capacitive load, while typically slewing at 1000 V/μs. The external compensation of AD8460 allow unlimited capacitive load drive. The AD8460 operates from high voltage dual supplies of ±50 V and a single low voltage supply of +5 V. Asymmetrical supply operation is available via full-scale adjustment.

Building the HDL project
------------------------

ADI does not distribute the bit/elf files of these projects so they must be built from the sources available :git-hdl>`__. To get the source you must `clone <https::`here </git-scm.com/book/en/v2/Git-Basics-Getting-a-Git-Repository>` the HDL repository.

Then go to the **\*PROJECT LOCATION WITHIN HDL (EX: projects/ad8460/zed)**\ \* location and run the make command by typing in your command prompt:

**Linux/Cygwin**

::

   user@analog:~$ cd hdl/projects/ad8460/zed
   user@analog:~/hdl/projects/ad8460/zed$ make

| Check :doc:`this guide </wiki-migration/resources/tools-software/linux-software/kuiper-linux>` on how to prepare your SD card with the proper boot files.
| A more comprehensive build guide can be found in the :doc:`HDL User Guide </wiki-migration/resources/fpga/docs/hdl>`.
