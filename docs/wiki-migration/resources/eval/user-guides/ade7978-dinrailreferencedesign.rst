Three Phase Din Rail Submeter Reference Design
==============================================

Overview
--------

This reference design uses the ADE7978/ADE7932 chipset in a modular form factor demonstrating the usage of thisi product line as a din rail mounted submeter.

Support
-------

Analog Devices will provide **limited** online support for anyone using the reference design with Analog Devices components via the :ez:`EngineerZone <community/energy-metering>` forum.

It should be noted, that the older the tools' versions and release branches are, the lower the chances to receive support from ADI engineers.
