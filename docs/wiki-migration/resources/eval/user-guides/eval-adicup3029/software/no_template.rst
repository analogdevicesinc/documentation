-----------------------------

The software pack can be downloaded in several ways.

-  Downloaded via the tools program

   -  It is **recommended** to download the software pack through from the tools program you are using. That way, all the files, directories structure, and project structure for the various applications is properly saved and accessed. For a detailed description on how to download the software packs through CrossCore Embedded Studio please see our :doc:`CrossCore Embedded Studio Quickstart User Guide </wiki-migration/resources/eval/user-guides/eval-adicup3029/tools/cces_user_guide>`.
