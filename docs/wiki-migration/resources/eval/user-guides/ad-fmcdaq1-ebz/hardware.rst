Size
----

The size of the board (not including the SMA connectors, which project beyond the edge of the board) is 3.102 inches by 2.712 inches (or about 79mm x 69mm).

.. image:: https://wiki.analog.com/_media/resources/eval/user-guides/ad-fmcdaq1-ebz/daq1_dimensions.png
