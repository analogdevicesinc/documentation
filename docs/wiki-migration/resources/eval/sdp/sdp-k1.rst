Preface
-------

The SDP-K1 is used as part of the evaluation system for many ADI components. The SDP-K1 board is designed to be used in conjunction with various ADI component evaluation boards as part of a customer evaluation environment. The SDP provides `USB <https://en.wikipedia.org/wiki/USB>`__ connectivity through a USB 2.0 high speed connection to the computer allowing users to evaluate components on this platform from a PC application. The SDP-K1 is based on an STM32F469NIH6 Cortex-M4 microcontroller, with the processor peripheral communication lines available to the component daughter board through the 120-pin small footprint SDP connector or the arduino header.

Product Overview
----------------

Board Features
~~~~~~~~~~~~~~

-  Arduino Uno headers
-  120-pin SDP connector
-  High performance ARM® Cortex™-M4 Core (STM32F469NI)
-  180 MHz max CPU frequency, 384 KB SRAM, 2048 KB Flash
-  1.8V or 3.3V selectable IO voltage
-  480Mbps High-Speed USB 2.0 (Device)
-  45MHz SPI
-  400KHz I2C
-  UART
-  Timers / PWM
-  12-bit ADC
-  GPIO
-  16MB SDRAM
-  3 traffic light LEDS (green, orange, red)
-  1 status LED

For more information, go to :adi:`The ADI web site <sdp>`.
