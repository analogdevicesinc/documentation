System Demonstration Platform Serial (SDP-S)
============================================

.. image:: https://wiki.analog.com/_media/resources/eval/sdp/sdp-s.png

Preface
=======

The SDP-S is part of the Analog Devices system demonstration platform (SDP). The SDP-S provides USB connectivity through a USB 2.0 high speed connection to the computer allowing users to evaluate components on this platform from a PC application. The SDP-S is based on a USB-to_serial engine, which has SPI, I\ :sup:`2`\ C and GPIO lines available, with a 120-pin small footprint connector.

Product Overview
================

The board features:

-  USB – to – serial engine
-  1 × 120-pin small footprint connector

   -  Hirose FX8-120P-SV1(91), 120-pin header

-  Peripherals exposed

   -  SPI
   -  TWI/I2C
   -  GPIO

For more information, go to :adi:`The ADI web site <sdp>`.
