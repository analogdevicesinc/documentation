MAX40109 Calibration Board User Guide
=====================================

General Description
-------------------

The MAX40109 calibration boards comes in two options, analog and digital output. The MAX40109CALSYSDG# calibration board supports the MAX40109 sensors with digital output (I2C). The MAX40109CALSYSAN# calibration board supports the MAX40109 sensors with analog output (0-5V, 4-20mA). The Windows®-compatible software provides a graphical user interface (GUI) that streamlines the calibration and supports up to 32 sensors per board.

Features
--------

Supports up to 32 Sensors

Windows GUI with USB Connection

Equipment Needed
----------------

1. Digital Calibration Board or Analog Calibration Board

2. Customer Sensor Board(s). Must have the MAX40109 installed and the Proper Signals Wired Out to the Respective Calibration Board.

3. USB to RS485 Cable with 4 loose Wires

4. Windows 10/11 PC
