Yocto Linux Examples
====================

This page summarises all the example applications that are available for the Yocto Linux for ADSP-SC5xx processors product.

-  :doc:`SHARC-ALSA Example </wiki-migration/resources/tools-software/linuxdsp/docs/quickstartguide/sharc-alsa-example>`: SHARC-ALSA is a framework designed to make a SHARC appear as an audio device in `ALSA <https://www.alsa-project.org/wiki/Main_Page>`__.
-  :doc:`USB Gadget Audio </wiki-migration/resources/tools-software/linuxdsp/docs/linux-kernel-and-drivers/usb/device_mode/gadget_audio_3_0_0>`: Turn the ADSP-SC5XX board into an audio card, using USB Gadget Audio Class 2.0
