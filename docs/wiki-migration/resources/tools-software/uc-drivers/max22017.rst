MAX22017 No-OS Driver
=====================

Supported Devices
-----------------

-  :adi:`MAX22017`

Overview
--------

The :adi:`MAX22017` is a two-channel industrial-grade configurable analog output device that can be changed on-the-fly in software to voltage or current output mode.
