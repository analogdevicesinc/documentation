Install the following tools on your computer

-  `STM32CubeIDE <https://www.st.com/en/development-tools/stm32cubeide.html>`__

-  `STM32CubeMX <https://www.st.com/en/development-tools/stm32cubemx.html>`__

-  `Git <https://git-scm.com/downloads>`__
-  `Make <https://gnuwin32.sourceforge.net/packages/make.htm>`__

--------------

Add the path "GnuWin32/bin" to the environment variables. For this, make sure to install 'make' in the root of 'C' drive without any spaces in the installation path. The path must be C:\\GnuWin32\\…
