.. important::

   This code has been developed and tested for mbed platform on the SDP-K1 and for STM32 platform on the Nucleo-L552ZEQ Controller Board. However, same code can be used without or with little modifications on any Mbed Or STM32 enabled board which has Arduino/Arduino+Zio Header Support on it, such as STM32-Discovery, STM32-Nucleo, etc.
