New and Experimental Things for the Sharc Audio Module
======================================================

Overview
--------

In addition to the :doc:`Baremetal Framework </wiki-migration/resources/tools-software/sharc-audio-module/baremetal>` as a software resource for experimenting with the SC589, a more advanced software project exists that allows users to quickly connect various audio inputs and outputs during run-time, giving them the capability to not only experiment with different audio-related technologies, but to also modify and add their own custom settings that may aid in rapid prototyping of their own systems.
