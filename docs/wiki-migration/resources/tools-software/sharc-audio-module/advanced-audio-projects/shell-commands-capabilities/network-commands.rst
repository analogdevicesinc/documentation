Overview
========

This section describes each of the networking related shell commands in detail.

Network Configuration commands
==============================

eth
---

+-------------------+---------------------------------------------------------------------------------------------------------------------------+
| **Description**   | Report ethernet status                                                                                                    |
+-------------------+---------------------------------------------------------------------------------------------------------------------------+
| **Inputs**        | None                                                                                                                      |
+-------------------+---------------------------------------------------------------------------------------------------------------------------+
| **Outputs**       | Displays network status information including hostname, IP Address, Gateway IP address, Netmask, MAC Addr and Link Status |
