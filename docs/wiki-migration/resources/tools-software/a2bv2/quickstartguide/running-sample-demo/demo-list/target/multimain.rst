Running sample demo: Multi main example
=======================================

-  Perform the hardware setup as per :doc:`ADZS2435-MINI Hardware modifications </wiki-migration/resources/tools-software/a2bv2/quickstartguide/hardware-setup/ad2435-with-som-carrier-ez-kit>` , :doc:`ADZS2435-MINI Jumper settings </wiki-migration/resources/tools-software/a2bv2/quickstartguide/hardware-setup/ad2435-with-som-carrier-ez-kit>` , :doc:`EVAL-AD2435WJ3LZ - sub 0 and sub-1 Jumper settings </wiki-migration/resources/tools-software/a2bv2/quickstartguide/hardware-setup/ad243x-high-power>`. Maintain the hardware configuration for the :doc:`SC594 </wiki-migration/resources/tools-software/a2bv2/quickstartguide/hardware-setup/ad2435-with-som-carrier-ez-kit>` and SOM-CRR ez kit as per the recommended default configurations of their respective manuals.
-  The default adi_a2b_multimainbusconfig.c can be used or it can be exported form "C:\\Analog Devices\\ADI_A2B-SSPlus_Software-Relx.y.z\\Schematics\\SC59x\\adi_a2b_ADZS2435MINI_SC59x_Multi_Main.ssprj"
-  Refer :doc:`Multimain Connections </wiki-migration/resources/tools-software/a2bv2/quickstartguide/hardware-setup/ad2435-with-som-carrier-ez-kit>` for setup.
-  Refer `running sample demo: multi main <https://wiki.analog.com/resources/tools-software/a2bv2/quickstartguide/running-sample-demo/demo-list/multimain>`_ for steps to run multi main example.
