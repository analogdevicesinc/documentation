Bus_Properties
==============

Bus_Properties has following attributes as shown below.

.. image:: https://wiki.analog.com/_media/resources/tools-software/a2bv2/a2bssplususerguide/exportinga2bsystemconfigurationfiles/image4.png
   :align: center

-  Num_of_Master_Nodes – Total no. of main nodes in the A2B Network.
-  Num_of_Slave_Nodes – Total no. of slave nodes in the A2B Network.

Each attribute is of “Byte” Data type.
