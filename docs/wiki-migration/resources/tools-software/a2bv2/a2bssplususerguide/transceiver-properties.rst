Transceiver properties
======================

The transceiver settings define how each node behaves in the network, including its communication roles, authentication mechanisms, and audio routing capabilities. This section provides a detailed breakdown of the transceiver configuration process, organized into three key areas: :doc:`General Settings </wiki-migration/resources/tools-software/a2bv2/a2bssplususerguide/transceiver-properties/transceiver-setting>`,\ :doc:`Exclude node setting </wiki-migration/resources/tools-software/a2bv2/a2bssplususerguide/a2bchainselection>` , :doc:`Custom Node Authentication </wiki-migration/resources/tools-software/a2bv2/a2bssplususerguide/transceiver-properties/customnode-auth>`, and :doc:`Crossbar Specifics </wiki-migration/resources/tools-software/a2bv2/a2bssplususerguide/transceiver-properties/crossbar>`.
