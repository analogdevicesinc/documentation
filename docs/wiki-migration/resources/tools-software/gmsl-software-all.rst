Requests
--------

If you find a ADI GMSL SerDes part that you would like a driver for, please send a request mail @gmsl_sw_request. This does go through a quick internal process before provide the software package or initiate the development. If you have a local ADI contact (like a Sales Engineer or FAE) it's always a great idea to contact them at the same time so we can better understand your application and schedule.
