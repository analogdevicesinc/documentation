Release Notes for FreeRTOS Addin 1.0.0 Product
==============================================

Introduction
------------

This document contains the release notes for FreeRTOS Addin for Analog Devices Processors version 1.0.0.

Supported processors:

-  ADSP-BF70x using CrossCore Embedded Studio (proprietary compiler)
-  ADSP-SC5xx (ARM Core) using CrossCore Embedded Studio (GCC)
-  ADSP-SC5xx (SHARC+ Core) using CrossCore Embedded Studio (proprietary compiler)
-  ADSP-21569 using CrossCore Embedded Studio (proprietary compiler)

Detailed instructions on supported versions of tools, and installation instructions can be found in the User Guide which is included in the zip file and is also available from the FreeRTOS page on :adi:`analog.com <en/index.html>`

--------------
