| =====Alternate Canvas===== SigmaStudio+ tool supports the option of creating alternate canvas of the current canvas. It is possible to swap the contents of the current active canvas with the contents of alternate canvas. These actions can be achieved through creating alternate canvas from context menu. Once the alternate canvas is created the contents of the current canvas is backed-up in alternate canvas.

| 
| Right click on the shape and click on the option to create alternate canvas.
| |image1|
| Give an appropriate name for the alternate canvas.
| |image2|
| Alternate canvas will appear in the project window.
| |image3|
| Current canvas can be switched with Alternate canvas as shown below.
