Navigate through a project
==========================

--------------

This section provides different ways of accessing canvas and settings windows in a project.

--------------

| Following are the different ways of accessing canvas and settings windows in a project:
| 1. Users can double click on the platform to open its canvas or right click on the platform and open its settings and canvas (refer the image below).
| |image1|
| 2. Users can open the processor's settings and the canvas from the project window (refer the image below).
| |image2|
| 3. Users can open the processor's settings and the canvas from the schematic window using the settings and canvas icon (refer the image below).
| |image3|
