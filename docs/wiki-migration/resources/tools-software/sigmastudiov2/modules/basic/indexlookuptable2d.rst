Index LookUp Table 2D
=====================

| 
| |image1|
| |image2|

Description
-----------

| The Index Lookup Table block lets you access values stored in a lookup table (LUT). There is a matrix of values in the table. The size of the table is user determined. There are two inputs, one for the X index and one for the Y index. In the above given example of 3x2 table To use this block drag it into the workspace and click 'OpenGrid' to open the Table Editor window
