IIR Coefficient (Direct Coefficient Entry)
==========================================

IIR Coefficient is an option on the :doc:`General 2nd Order Filter </wiki-migration/resources/tools-software/sigmastudio/toolbox/filters/general2ndorder>` algorithm that allows the user to manually enter IIR coefficients b0, b1, b2, a1, and a2.

Using the IIR Coefficient Option
--------------------------------

-  Add a General (2nd order) filter from the toolbox to the schematic.

|image1| |image2|

-  Click on the blue icon in the middle of the cell to access the options.

|image3| |image4|

-  In the drop-down menu, select IIR Coefficient.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudio/toolbox/filters/general2ndorder/screenhunter_05_jul._18_10.44.jpg

-  Manually enter in the coefficient values.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudio/toolbox/filters/general2ndorder/screenhunter_06_jul._18_10.45.jpg

-  Click the Load button to download the coefficients.

.. image:: https://wiki.analog.com/_media/resources/tools-software/sigmastudio/toolbox/filters/general2ndorder/screenhunter_07_jul._18_10.46.jpg

-  You can use the :doc:`Stimulus </wiki-migration/resources/tools-software/sigmastudio/toolbox/systemschematicdesign/simulationstimuli>` and :doc:`Probe </wiki-migration/resources/tools-software/sigmastudio/toolbox/systemschematicdesign/simulationprobe>` cells to check the magnitude and phase responses of the filter.
