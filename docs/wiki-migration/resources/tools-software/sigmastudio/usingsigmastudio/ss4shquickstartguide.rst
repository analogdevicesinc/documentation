| ======SigmaStudio for SHARC - Quick start guide====== SigmaStudio™ is a development environment from Analog Devices for graphically programming ADI’s DSPs. SigmaStudio for SHARC includes an extensive set of algorithms to perform audio processing tasks such as filtering and mixing, as well as basic low-level DSP functions, optimized to run on the SHARC family of processors. The environment is integrated with CrossCore® Embedded Studio.
| This document explains how to use SigmaStudio for SHARC (ADSP-SC5xxi). This document is a Quick Start Guide to getting a SigmaStudio for SHARC demonstration working on ADSP-SC5xx processors. Developers are assumed to be familiar with SigmaStudio. A basic understanding of the SHARC EZ-BOARD is recommended.
