ADALM2000 Quick Start
=====================

Most people don't read `instructions <https://wiki.analog.com/../users>`_, and just want get things going. This is the minimum to get things going, and a few common trouble shooting tips along the way.

-  install the drivers

   -  :doc:`Windows </wiki-migration/university/tools/pluto/drivers/windows>` or :doc:`Linux </wiki-migration/university/tools/pluto/drivers/linux>` or :doc:`MAC </wiki-migration/university/tools/pluto/drivers/osx>`

-  plug the device in to USB, and check things are working

   -  :doc:`Windows </wiki-migration/university/tools/pluto/drivers/windows>` or :doc:`Linux </wiki-migration/university/tools/pluto/drivers/linux>` or :doc:`MAC </wiki-migration/university/tools/pluto/drivers/osx>`

-  install :doc:`Scopy </wiki-migration/university/tools/m2k/scopy>`
