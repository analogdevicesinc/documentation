EVAL-ADAQ23875-FMCZ
===============================================================================

16-Bit, 15 MSPS, High Speed, Precision µModule Data Acquisition Solution.

Overview
-------------------------------------------------------------------------------

The :adi:`EVAL-ADAQ23875FMCZ` evaluation board enables simplified performance
assessment of the :adi:`ADAQ23875`, a 15 MSPS, 16-bit, High Speed, Precision
µModule Data Acquisition Solution. The µModule integrates passive components
with superior matching and drift characteristics to minimise temperature-
dependent error sources and optimise performance for high channel count
multiplexed architectures.

The evaluation board comprises one µModule (:adi:`ADAQ23875`), a choice of
4.096 V reference (:adi:`LTC6655`) or 2.048 V reference (:adi:`ADR4520`),
on-board power supplies using :adi:`LTM8049`, :adi:`ADP7118`, :adi:`ADP7183`,
:adi:`LT3023`, and an 800 MHz clock distribution IC. Optional :adi:`ADA4899-1`
amplifiers are available for the signal conditioning path.

.. image:: images/eval-adaq23875fmcz-top.jpg
   :align: center
   :width: 500

Features:

- Evaluation board for the :adi:`ADAQ23875` 16-bit, 15 MSPS µModule ADC
- Integrated fully differential ADC driver with signal scaling
- On-board reference buffer with VCMO generation
- PC software for control and data analysis of time and frequency domain
