| =====Restrict/Un-Restrict Shape===== SigmaStudio+ tool supports the option of Restrict shape from the Context Menu or the Properties Window. This is to avoid accidental changes to a particular configured shape by the user. The user can easily Un-Restrict the shape by right-clicking on the shape or clicking the checkbox from the properties window.
