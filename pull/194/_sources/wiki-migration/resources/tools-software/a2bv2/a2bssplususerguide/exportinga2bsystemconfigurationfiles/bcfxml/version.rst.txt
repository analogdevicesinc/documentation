Version_Info
============

Version_info has following sub xml elements as shown below.

.. image:: https://wiki.analog.com/_media/resources/tools-software/a2bv2/a2bssplususerguide/exportinga2bsystemconfigurationfiles/image3.png
   :align: center

-  BCF_Version – User defined Bus Config File version exported from SigmaStudio / SigmaStudio+
-  A2BDLL_Version – Version of A2B DLL used for schematic creation.
-  A2BStackDLL_Version – Version of A2B Stack used for compiling the A2B Schematic.
-  SigmaStudio_Version – Version of SigmaStudio / SigmaStudio+ used for generating bus config file.

Each sub element has “Version” attribute with “String” Data type respectively.
