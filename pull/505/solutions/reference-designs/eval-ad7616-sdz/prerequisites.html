Prerequisites

What you need depends on which carrier platform you are targeting. As a minimum, you need to start out with:

Hardware prerequisites

  1. The EVAL-AD7616SDZ evaluation board

  2. A carrier platform — see the table below for supported options

    Carrier

    Interface to EVAL-AD7616

    Additional hardware required

    ZedBoard

    FMC

    SDP-I-FMC adapter board

    ZC706

    FMC

    SDP-I-FMC adapter board

    SDP-K1

    Fly-wire (SPI)

    STLINK-V3 programmer

  3. Power supplies:

    • Both the EVAL-AD7616SDZ and the carrier must be powered via their respective connectors.

Software prerequisites

  1. Xilinx Vivado and Vitis — the supported version can be found on the HDL releases page. Required when targeting ZedBoard or ZC706 (FPGA-based flow).

  2. A UART terminal (e.g. Tera Term or PuTTY):

    • ZedBoard / ZC706: baud rate 115200 (PS UART console)

    • SDP-K1: baud rate 230400 (IIO serial interface)

  3. STLINK-V3 drivers — required when using the SDP-K1 carrier to flash the firmware.

  4. IIO Oscilloscope (optional) — can be used to visualize the captured data when running the No-OS IIO project on the SDP-K1.
