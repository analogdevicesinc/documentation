Jupiter SDR - known issues

Jupiter SDR known issues when used with 2023_r2 Patch1 Kuiper image

  1. IIO buffer size limited to 131072 samples when USB 3 interface is used to stream data to a host

  2. USB 3 and Gigabit Ethernet interfaces throughput is currently limited by libiio v0.26 implementation. The throughputs will significantly improve once libiio v1.0 will be realeaed.

  3. IIO Oscilloscope profile generator (latest libadrv9002-iio) supports v68.10.1 API while 2023_r3 Patch1 comes with v68.14.10 API; Generate a custom device profile using TES

  4. Multi chip synchronization support is not included in the 2023_r2 Patch1 Kuiper release. Please visit multi-chip synchronization page to check available support.

  5. ADRV9002 CMOS interface not implemented
