Prerequisites

What you need depends on what you are trying to do. As a minimum, you need to start out with:

Hardware prerequisites

  1. AD9208-DUAL-EBZ FMC evaluation board

  2. An FPGA carrier platform: AMD Xilinx VCU118 (Virtex UltraScale+)

  3. 2x Micro-USB cable

  4. Ethernet cable

  5. Signal generator

  6. Low phase noise analog input source and antialiasing filter

  7. 4-way splitter (optional)

Note

For standalone single-chip evaluation using ACE software on ADS7-V2EBZ or ADS8-V1EBZ, see the AD9208-3000EBZ evaluation guide for specific equipment requirements.

Software prerequisites

  1. A Linux OS on a PC

  2. Xilinx Vivado 2019.1 or later

  3. UART terminal (e.g. TeraTerm, PuTTY, Minicom)

  4. IIO Oscilloscope

Helpful documents

  1. AD9208 Datasheet

  2. AD9689 Datasheet

  3. AD9699 Datasheet

  4. AN-905 Application Note, VisualAnalog Converter Evaluation Tool Version 1.0 User Manual

  5. AN-835 Application Note, Understanding ADC Testing and Evaluation

  6. JESD204B Survival Guide

Note

ADI does not offer FPGA carrier platforms for sale or loan; getting one yourself is the normal part of development or evaluation.
