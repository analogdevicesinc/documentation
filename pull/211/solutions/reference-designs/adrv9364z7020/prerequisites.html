Prerequisites

What you need depends on what you are trying to do. As a minimum, you need to start out with:

  1. The ADRV9364-Z7020 SOM, which integrates a Xilinx Zynq®-7020 SoC and the AD9364 RF transceiver.

  2. A carrier board for the SOM. The supported carriers are:

    • ADRV1CRR-BOB — Break-Out Board carrier (CMOS or LVDS mode)

    • ADRV1CRR-PACKRF PackRF carrier, LVDS mode (OBSOLETE)

  3. Some way to interact with the platform, which normally includes:

    • HDMI monitor

    • USB Keyboard

    • USB Mouse

  4. Internet connection (without proxies makes things much easier) to update the scripts/binaries on the SD card (firewalls are OK, proxies make things a pain).

  5. RF test equipment.

  6. An SD card with at least 16GB of memory imaged with Kuiper Linux.
