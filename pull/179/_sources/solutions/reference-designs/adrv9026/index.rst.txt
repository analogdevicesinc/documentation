ADRV9026 User Guide
====================

Introduction
------------

The :adi:`ADRV9026` is a highly integrated RF agile transceiver offering
four independently controlled transmitters, dedicated observation receivers
for monitoring each transmitter output, four independently controlled
receivers, integrated synthesizers, and digital signal processing functions.

The EVAL-ADRV9026/ADRV9029 evaluation boards are FMC mezzanine cards designed
for use with compatible FPGA development boards. The :adi:`ADRV9029` uses the
same driver infrastructure and evaluation board design.

Prerequisites
~~~~~~~~~~~~~

- EVAL-ADRV9026 or EVAL-ADRV9029 evaluation board
- One of the supported carrier boards
- DisplayPort monitor, USB keyboard and mouse (for ARM/FPGA SoC platforms)
- LAN cable and host PC (for FPGA-only solutions)
- Internet connection for updating SD card contents
- RF test equipment

Hardware Setup
~~~~~~~~~~~~~~

The EVAL-ADRV9026/ADRV9029 boards connect to the HPC1 FMC connector (unless
otherwise noted). The carrier setup requires:

- Power: 12 VDC, 3 A max
- UART: 115200 baud
- Ethernet: for Linux
- HDMI/DisplayPort: if available
- JTAG: for no-OS

HDL Reference Design
--------------------

The HDL reference design is an embedded system built around a processor core.
The device digital interface is handled by the transceiver IP followed by the
JESD204B and device-specific cores. The cores are programmable through an
AXI-Lite interface. The delineated data is then passed on to independent DMA
cores for the transmit and receive paths.

IIO Devices
~~~~~~~~~~~

The following IIO devices are enumerated on a ZCU102 system:

- **ad9528-1** - Clock generator
- **adrv9025-phy** - Transceiver control
- **axi-adrv9025-rx-hpc** - Receive data path (buffer capable)
- **axi-adrv9025-tx-hpc** - Transmit data path (buffer capable)

The main receivers RX1-RX4 are handled by the ``axi-adrv9025-rx-hpc`` IIO
device with channels: voltage0_i/q through voltage3_i/q.
