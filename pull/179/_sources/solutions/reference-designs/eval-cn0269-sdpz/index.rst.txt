EVAL-CN0269-SDPZ
=================

Precision Sigma-Delta ADC Measurement System.

Overview
--------

:adi:`CN0269` is a precision measurement system based on the :adi:`AD7705`
sigma-delta ADC with integrated signal conditioning for industrial sensor
applications.

The circuit uses the :adi:`AD7705` 16-bit sigma-delta ADC with programmable
gain amplifier and low-power operation.

Required Equipment
------------------

- EVAL-CN0269-SDPZ evaluation board
- :adi:`EVAL-SDP-CB1Z` controller board (SDP-B board)
- ±15 V dual power supply
- USB cable

Hardware Setup
--------------

- Connect the EVAL-CN0269-SDPZ to the SDP-B board via the 120-pin connector.
- Apply ±15 V power to the designated power terminals.
- Connect USB cable from SDP-B board to PC.

Software Setup
--------------

The evaluation software provides data acquisition, configuration, and data
export capabilities through a graphical user interface.
