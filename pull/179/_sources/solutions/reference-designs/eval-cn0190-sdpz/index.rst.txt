EVAL-CN0190-SDPZ
=================

Multivoltage Power System Reference.

Overview
--------

:adi:`CN0190` is a reference solution for multivoltage power systems with 12
output power rails, delivering approximately 78% overall efficiency with about
25 W output power. The system demonstrates overvoltage, undervoltage, and
overcurrent detection/protection, sequencing, and power margining control.

The circuit uses the :adi:`ADM1178` for overvoltage/overcurrent detection and
hot-swap control, and the :adi:`ADM1066` for power supply monitoring and
sequencing control across all 12 rails. Multiple switching regulators and LDOs
provide output voltages ranging from +3.3 V to ±15 V.

Required Equipment
------------------

- EVAL-CN0190-EB1Z evaluation board
- DC power supply: 6--14 V input
- Oscilloscope (e.g., Tektronix TDS3034B) for sequencing verification
- Electronic load (e.g., Agilent N3302A)
- Digital multimeter
- USB cable for programming interface

Hardware Setup
--------------

- **Connector 1** -- Input: 6--14 V DC.
- **Connectors 2--13** -- 12 output power rails with various topologies
  (synchronous buck, LDO, inverted buck/boost, Sepic-Cuk).
- **Connector 15** -- I2C programming interface (SCLK, SDA, GND).
- Output voltages range from +3.3 V (2 A) to ±15 V (0.1 A).
- ADM1066 comes pre-programmed with all outputs active simultaneously.

Software Setup
--------------

The SuperSequencer evaluation software and prog106x application software
provide ADM1066 EEPROM programming and custom sequencing configuration via
I2C interface.
