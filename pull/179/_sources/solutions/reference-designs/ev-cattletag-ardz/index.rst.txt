EV-CATTLETAG-ARDZ
==================

Solar-Powered Livestock Tracking Sensor.

Overview
--------

The EV-CATTLETAG-ARDZ is an Arduino-compatible sensor board with a fully
integrated solution for harvesting energy from single-/multi-cell solar sources
with integrated battery charging and temperature sensing for livestock tracking
applications. The board combines the :adi:`MAX20361` power harvester, the
:adi:`MAX20335` battery management system (BMS), and the :adi:`MAX30210` digital
temperature sensor with a buzzer and optional vibration motor.

.. figure:: ev-cattletag-ardz.jpg
   :width: 600 px
   :align: center

   EV-CATTLETAG-ARDZ Evaluation Board

The EV-CATTLETAG-ARDZ is designed to work with the
:ref:`MAX32670-LR-ARDZ <ad-max32lrwise-sl>` base board as part of the ADI
Long Range Wireless Radio solution, enabling proprietary long-range
communication techniques suitable for livestock monitoring applications.

Applications include:

- Real-time livestock health monitoring, enabling rapid treatment intervention
- Grazing animal location tracking to prevent loss and analyze grazing patterns
- Historical health trend analysis and disease tracking
- Breeding readiness and birth monitoring to optimize reproduction
- Asset location, recovery, traceability, inventory management, and theft
  prevention

Power Supply Options
--------------------

The EV-CATTLETAG-ARDZ supports multiple power source configurations.

.. figure:: ev-cattletag-ardz_power_source_option.png
   :width: 700 px
   :align: center

   Power Source Options

.. list-table:: Power Source Configurations
   :header-rows: 1

   * - Source
     - Connector
     - Description
   * - Solar Panel
     - P6
     - Solar panel input used to recharge the lithium-ion battery connected to
       P8 or the 2x AAA battery holder
   * - External Power
     - P8
     - External power supply terminal block, 3 V to 4.2 V allowable input
   * - 2x AAA Battery
     - Battery Holder
     - Onboard 2x AAA battery holder
   * - Arduino Host Power
     - Arduino Headers
     - Power supplied directly from the host base board

.. figure:: ev-cattletag-ardz_power_supply_input.png
   :width: 700 px
   :align: center

   Power Supply Input Connectors

Solar Panel Configuration
~~~~~~~~~~~~~~~~~~~~~~~~~

The board includes an onboard solar panel input through the P6 terminal block.
When using an external power source connected to P6, it is required to
disconnect the onboard solar panel by removing resistor **R30**.

.. figure:: ev-cattletag-ardz_solar_panel_option.png
   :width: 700 px
   :align: center

   Solar Panel Option Configuration

Arduino Host Power Configuration
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

When power is supplied directly from the host base board through the Arduino
headers, it is required to remove resistor **R84** and place a 0 ohm resistor
at **R86**.

Arduino Connector Interface
----------------------------

The EV-CATTLETAG-ARDZ uses the standardized Arduino header interface, providing
support for various digital communication protocols such as SPI, I2C, and UART.
The board is designed to connect directly to the
:ref:`MAX32670-LR-ARDZ <ad-max32lrwise-sl>` base board via these headers.

   :width: 700 px
   :align: center

   Arduino Connector Interface

Required Equipment
------------------

- EV-CATTLETAG-ARDZ sensor node
- :ref:`MAX32670-LR-ARDZ <ad-max32lrwise-sl>` base board
- MAX32625PICO programming adapter with 10-pin ribbon cable
- CR123A battery (3 V to 4.7 V) or equivalent external DC supply
- Micro-USB to USB cable
- Host PC (Windows 10 or later)

Hardware Setup
--------------

.. figure:: hardware_setup.png
   :width: 600 px
   :align: center

   Complete Hardware Setup

Step 1 -- Insert Battery
~~~~~~~~~~~~~~~~~~~~~~~~

Insert a CR123A battery into the BT1 connector on the
:ref:`MAX32670-LR-ARDZ <ad-max32lrwise-sl>` base board. Verify correct polarity;
the **DS3** LED illuminates to confirm correct installation.

.. figure:: max32670-lr-ardz_with_battery.png
   :width: 500 px
   :align: center

   MAX32670-LR-ARDZ Base Board with CR123A Battery Installed

Step 2 -- Connect Sensor Node
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Align the Arduino headers and connect the EV-CATTLETAG-ARDZ sensor node to the
MAX32670-LR-ARDZ base board.

Step 3 -- Connect Programmer
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Connect the MAX32625PICO programming adapter to the base board via the 10-pin
ribbon cable. Ensure the correct orientation of the ribbon cable connector.

.. figure:: max32670-lr-ardz_to_maxpico.png
   :width: 600 px
   :align: center

   MAX32670-LR-ARDZ Connected to MAX32625PICO Programmer

Step 4 -- Connect to Host PC
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Connect the MAX32625PICO to the host PC using a micro-USB to USB cable.

Software Setup
--------------

Programming the MAX32625PICO
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The MAX32625PICO requires a specific firmware image to program the MAX32670
microcontroller on the base board. The required firmware image is
``max32625_max32670evkit_if_crc_swd_v1.0.3.bin``
(MAX32625PICO Firmware Image for MAX32670, v1.0.3 SWD).

.. figure:: max32625pico_maxdap.png
   :width: 400 px
   :align: center

   MAX32625PICO Programming Adapter

Follow these steps to load the firmware:

#. Do not connect the MAX32625PICO to the MAX32670-LR-ARDZ base board yet.
#. Connect the MAX32625PICO to the host PC using a micro-USB to USB cable.
#. Press and hold the button on the MAX32625PICO. **Do not release the button
   until the MAINTENANCE drive is mounted.**
#. Release the button once the MAINTENANCE drive appears.
#. Drag and drop the firmware image file to the MAINTENANCE drive.
#. After a few seconds, the MAINTENANCE drive will disappear and be replaced
   by a drive named **DAPLINK**, confirming successful firmware installation.
