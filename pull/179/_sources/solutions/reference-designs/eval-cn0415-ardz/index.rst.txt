EVAL-CN0415-ARDZ
=================

PWM Solenoid Driver.

Overview
--------

:adi:`CN0415` is a robust pulse-width modulation (PWM) solenoid driver circuit
in Arduino shield form factor. The system provides accurate, closed-loop current
control for 2-state solenoid applications such as motion control, latches,
clamps, brakes, clutches, and proportional valves.

The circuit uses the :adi:`LTC4367` supply supervisor, :adi:`LTC4441` gate
driver, :adi:`LTC2312-14` 14-bit ADC, :adi:`LT1671` op-amp, :adi:`LT3433`
switching regulator, :adi:`ADG3304` logic level translator, and the
:adi:`AD8210` current sense amplifier.

Required Equipment
------------------

- EVAL-CN0415-ARDZ evaluation board (Arduino shield)
- :adi:`EVAL-ADICUP3029` development board
- 6--24 V power supply (up to 3 A)
- Solenoid actuator (24 V max, 3 A max)
- USB cable
- PC with serial terminal software

Hardware Setup
--------------

- **DS1--DS4** -- LED indicators for power and fault detection.
- **P2** -- Chip select configuration (GPIO pins 6, 2, 1).
- **P34** -- PWM selection (GPIO pins 7-7, 7-6, 7-4).
- **JP1** -- Current monitor offset.
- **JP2** -- ADC selection.
- **JP3** -- Vcc supply configuration.
- **JP5** -- Burst-mode enable.
- **JP8** -- Overcurrent latch-off control.
- **P6, P7** -- Hardware fault detection pins.

Software Setup
--------------

The command-line interface (CLI) via serial UART provides independent control
of initial pull-in current, pull-in time delay, and hold current. PWM duty
cycle and frequency are directly adjustable, and programmable dither current
reduces mechanical stiction.
