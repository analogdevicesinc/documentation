AD9783-EBZ User Guide
=====================

Introduction
------------

The :adi:`AD9783` includes pin-compatible, high dynamic range, dual
digital-to-analog converters (DACs) with 16-bit resolution and a sample rate
of up to 500 MSPS. The device includes specific features for direct conversion
transmit applications, including gain and offset compensation, interfacing
seamlessly with analog quadrature modulators.

The reference design consists of a DDS module and an LVDS interface. The
AD9783-EBZ evaluation board connects to the carrier through a DAC-FMC
interposer board.

This evaluation guide describes both the hardware and software setup needed
to characterize the :adi:`AD9780`/:adi:`AD9781`/:adi:`AD9783` 12-/14-/16-bit,
500 MSPS high-speed digital-to-analog converters using the AD9783-DPG2-EBZ
evaluation board with the ADS7-V2 or SDP-H1 controller boards.

Supported Devices
-----------------

- :adi:`AD9780`
- :adi:`AD9781`
- :adi:`AD9783`

Supported Carriers
------------------

- :xilinx:`ZCU102 <products/boards-and-kits/ek-u1-zcu102-g.html>` (via DAC-FMC interposer board)

Evaluation Setup
----------------

Equipment Needed
~~~~~~~~~~~~~~~~

- AD9783-DPG2-EBZ evaluation board
- SDP-H1 (EVAL-SDP-CH1Z) board or ADS7-V2 (ADS7-V2EBZ) board
- :adi:`AD-DAC-FMC-ADP <AD-DAC-FMC>` high-speed DAC evaluation board to FMC
  adapter board
- 5 Vdc, 1 A power supply for the evaluation board
- High-frequency continuous wave generator (clock source)
- Signal/spectrum analyzer
- 2 x SMA cables
- 2 x banana plug cables
- PC with ACE and DPG Lite software

**Additional items for SDP-H1 evaluation kit:**

- 12 Vdc, 1 A wall adapter
- 2 x USB-A to USB-Mini cables

**Additional items for ADS7-V2 evaluation kit:**

- 12 Vdc power supply with power cord
- USB-A to USB-B cable

Software Needed
~~~~~~~~~~~~~~~

- `Analysis | Control | Evaluation (ACE) <https://www.analog.com/en/resources/evaluation-hardware-and-software/evaluation-development-platforms/ace-software.html>`__
- DPG Lite (recommended; installed with ACE)

Quick Start Guide
-----------------

1. Attach the AD9783-DPG2-EBZ to the controller board's (SDP-H1 or ADS7-V2)
   FMC connector using the :adi:`AD-DAC-FMC-ADP <AD-DAC-FMC>` adapter board.
2. Connect the controller board (SDP-H1 or ADS7-V2) to the PC via USB and to
   a **12 V power supply**.
3. Connect the evaluation board to the PC via USB and to a **5 Vdc, 1 A power
   supply** on **P5** and **P6**. Refer to the setup figures above.
4. Connect a continuous wave generator for clock input to **SMA J1** and the
   DAC output from **SMA J5** (IOUT1P) or **SMA J9** (IOUT2P) to a
   signal/spectrum analyzer.
5. Set the clock input to **200 MHz**, **3 dBm**.
6. Open ACE. The board will be automatically recognized by the software.
   Click the plugin. If the board is not auto-detected, install the AD9783
   ACE plugin manually.

   .. figure:: ace_board_detect.png
      :align: center

      AD9783 plugin detected in ACE

7. On the ACE Board View, double-click the AD9783 block to access the
   Chip View.

   .. figure:: ace_gui.jpg
      :align: center

      AD9783 Chip View in ACE

8. The default register values are displayed. Values can be changed on the
   Chip View GUI or via the **Memory Map** for registers not available in the
   GUI. Once the desired values are set, click **Apply Changes** and then
   **Read All** in the upper-right window.

   .. figure:: ace_apply.png
      :align: center

      Apply Changes and Read All buttons

9. Start DPG Lite. The controller board (SDP-H1 or ADS7-V2), the device part
   number, and clock frequency should be displayed.

   .. figure:: ad9783_dpg_window_sdp-h1.png
      :align: center

      DPG Lite session for EVAL-AD9783 with SDP-H1

   .. figure:: ad9783_dpg_window_ads7v2.png
      :align: center

      DPG Lite session for EVAL-AD9783 with ADS7-V2

10. In DPG Lite, from the **Add Generator Waveforms** pulldown menu, select
    **Single Tone** and apply the following settings:

    - **Data Rate**: 200 MHz
    - **Desired Frequency**: 16 MHz
    - **DAC Resolution**: 16 for :adi:`AD9783`, 14 for :adi:`AD9781`,
      12 for :adi:`AD9780`
    - Uncheck **Unsigned Data**
    - Check **Generate Complex Data (I & Q)**

11. Select the I/Q tone from the **I/Q Data Vector** pulldown menu.
12. Press the download arrow and then the play button. A spectrum similar to
    the figure below should appear on the signal/spectrum analyzer.

    .. figure:: c_ad9783_16mhz.png
       :align: center

       Spectrum output for AD9783: Fdac = 200 MSPS, Fout = 16 MHz

Using the ADL5375 Modulator
~~~~~~~~~~~~~~~~~~~~~~~~~~~

By default, solder jumpers JP4, JP5, JP6, and JP7 are configured to route
the DAC outputs to SMA J5 (IOUT1P) and J9 (IOUT2P). To connect the DAC
output to the filter that feeds into the :adi:`ADL5375` modulator, reconfigure
the solder jumpers as shown in the right side of the figure below. Source the
desired LO frequency (e.g., 900 or 1800 MHz, 3 dBm) on SMA J2 (LO IN). The
ADL5375 modulator output can be observed on SMA J6 (RF OUT).

.. figure:: jumper_config.jpg
   :align: center

   Left: DAC output configuration. Right: Modulator output configuration.

Troubleshooting
---------------

If unexpected results occur, check the following:

- Verify the voltage rails of the evaluation board: P5 and P6 should be 5 V;
  TP3, TP5, and TP7 should be 3.3 V; TP9 and TP11 should be 1.8 V.
- Check that all three blue LEDs on the AD-DAC-FMC-ADP board are lit.
  Reconnect the board to the FMC connector of the controller board if the LEDs
  are not lit.
- Verify that the controller board (SDP-H1 or ADS7-V2) is being supplied
  properly by the 12 Vdc adapter. Some LEDs on the controller board should be
  lit.
- Perform a chip reset by pressing **S2** on the AD9783 evaluation board.
- Power cycle both the controller board and the AD9783 evaluation board.
- Probe **C18** and **C19** to verify that a clock signal is being sent to the
  DAC.
- Check that **XD1** is lit on the evaluation board. Reconnect or replace the
  USB cable if it is not lit.
- Disconnect and reconnect the controller board and AD9783 evaluation board.
  Reopen ACE and DPG Lite software.
