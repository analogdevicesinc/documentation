EVAL-CN0394-ARDZ
=================

Multi-Channel Thermocouple Temperature Meter.

Overview
--------

:adi:`CN0394` is a single-supply, multi-channel thermocouple temperature meter
with RTD cold junction compensation. The system supports B, E, J, K, N, R, S,
and T type thermocouples across 4 independent channels.

The circuit uses the :adi:`ADuCM360` low-power precision analog microcontroller
with dual sigma-delta ADCs and ARM Cortex-M3 core for thermocouple signal
processing and linearization.

Required Equipment
------------------

- EVAL-CN0394-ARDZ evaluation board (Arduino shield)
- :adi:`EVAL-ADICUP360` development board
- Up to 4 thermocouples (any combination of B, E, J, K, N, R, S, T types)
- USB cable
- PC with serial terminal software (PuTTY or Tera Term)

Hardware Setup
--------------

- Connect the EVAL-CN0394-ARDZ shield to the EVAL-ADICUP360 board.
- Connect thermocouples to P1--P4 (labeled CH0--CH3 on PCB) using Omega-style
  universal connectors.
- Connect the USER USB port on EVAL-ADICUP360 to PC.
- Built-in RTD beneath copper tabs provides cold junction temperature
  measurement.

Software Setup
--------------

The CN0394 demo firmware runs on the :adi:`EVAL-ADICUP360` platform and
provides multi-channel temperature readout via serial terminal interface.
