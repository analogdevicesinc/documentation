AD-FMCDAQ2-EBZ Bare Metal Quick Start Guide
=============================================

Xilinx Platform
---------------

This guide provides some quick instructions on how to setup the AD-FMCDAQ2-EBZ
on either:

- `ZC706 <https://www.xilinx.com>`__
- `KC705 <https://www.xilinx.com>`__
- `KCU105 <https://www.xilinx.com>`__
- `VC707 <https://www.xilinx.com>`__
