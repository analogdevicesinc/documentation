AD-FMCOMMS8-EBZ User Guide
============================

Introduction
------------

The :adi:`AD-FMCOMMS8-EBZ
<en/design-center/evaluation-hardware-and-software/evaluation-boards-kits/eval-ad-fmcomms8-ebz.html>`
is an integrated RF design containing two :adi:`ADRV9009` wideband
transceivers. By connecting to a compatible FPGA development board that
supports FMC HPC mechanical connector and JESD204B bus interface, it can be
used for evaluation and prototyping with up to 4 transmit and receive channels
that can be synchronized in phase and frequency.

It can also be used with the :ref:`ADRV9009-ZU11EG <adrv9009zu11eg>` RF-SOM
system for up to 8 phase and frequency synchronized transmit and receive
channels.

.. image:: fmcomms8_top.png
   :align: center
   :width: 400

Features
~~~~~~~~

- Two ADRV9009 devices providing:

  - Quad transmitters / Quad receivers
  - Quad input observation receiver for DPD
  - Max Rx BW: 200 MHz
  - Max tunable Tx synthesis BW: 450 MHz
  - Max observation Rx BW: 450 MHz
  - Tuning range: 75 MHz to 6000 MHz
  - Fully integrated fractional-N RF synthesizers
  - Multi-chip phase synchronization for all RF LO and baseband clocks

- Multi-chip phase synchronization using :adi:`HMC7044`
- FMC HPC compatible interface

Quick Start
-----------

Prerequisites
~~~~~~~~~~~~~

- AD-FMCOMMS8-EBZ evaluation board
- One of the supported carrier boards
- ADI Kuiper Linux SD card image
- Host PC with Ethernet connection for IIO Oscilloscope remote mode

Hardware Setup (ADRV9009-ZU11EG)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

When used with the ADRV9009-ZU11EG and ADRV2CRR-FMC carrier:

#. Connect the AD-FMCOMMS8-EBZ to the FMC HPC connector on the ADRV2CRR-FMC
#. Connect power supply (12 V, 145 W included with carrier kit)
#. Connect UART (115200 baud) and Ethernet
#. Insert SD card with Kuiper Linux image
#. Set boot mode switches for SD card boot (S13-S16: 1110)
#. Power on the system

Hardware Setup (Arria 10 SoC)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

When used with the Arria 10 SoC Development Kit, six zero-ohm resistors
must be moved on the AD-FMCOMMS8-EBZ board to accommodate the FMC pin
connections on the Arria 10 carrier.

Verifying Operation
~~~~~~~~~~~~~~~~~~~

After boot, verify the JESD204B link status:

.. code-block:: bash

   jesd_status

Verify clock chip lock status using ``iio_attr``:

.. code-block:: bash

   iio_attr -d hmc7044 status

Use IIO Oscilloscope in remote mode to capture and visualize data:

.. code-block:: bash

   # On host PC
   osc
