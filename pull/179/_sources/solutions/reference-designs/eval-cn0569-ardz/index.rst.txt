EVAL-CN0569-ARDZ
================

Infrared Light Angle Sensor Module for Gesture Recognition.

Overview
--------

The :adi:`EVAL-CN0569-PMDZ <CN0569>` is a compact evaluation board for the
CN0569 infrared light angle sensor module designed for gesture recognition
applications. The board measures 2.7 x 0.8 x 0.062 inches and uses a 4-layer
PCB construction.

The design uses the :adi:`ADPD1080` photometric front-end device with two
:adi:`ADPD2140` infrared photodetector sensors positioned 1 inch apart,
enabling dual-axis gesture detection. The board includes two 3D-printed
optical baffles for optimal sensor performance.

Features
~~~~~~~~

- I2C Pmod connector interface (1x6, 0.1-inch header)
- :adi:`ADPD1080` photometric front-end
- Two :adi:`ADPD2140` sensors for dual-axis gesture detection
- Operates from Pmod VCC (3.3 V typical)
- Internal 1.8 V regulated supply for :adi:`ADPD1080`
- Breakaway line for custom sensor spacing
- Supports four directional gestures (up, down, left, right)

I2C Pmod Connector Pinout
~~~~~~~~~~~~~~~~~~~~~~~~~~

+-------+----------+-----------------------+
| Pin   | Name     | Function              |
+=======+==========+=======================+
| 1     | GPIO0    | Interrupt (INT)       |
+-------+----------+-----------------------+
| 2     | NC       | Not connected         |
+-------+----------+-----------------------+
| 3     | SCL      | I2C Clock             |
+-------+----------+-----------------------+
| 4     | SDA      | I2C Data              |
+-------+----------+-----------------------+
| 5     | GND      | Ground                |
+-------+----------+-----------------------+
| 6     | VCC      | Power Supply (3.3 V)  |
+-------+----------+-----------------------+

Accessible Test Points
~~~~~~~~~~~~~~~~~~~~~~

The board exposes multiple test points for advanced applications:

- **LED Drivers**: LED1 (IR LED, default), LED2, LED3 for external LED
  integration
- **GPIO Access**: IO0 and IO1 pins (1.8 V logic levels; level shifting
  required for 3.3 V systems)
- **Power**: 1V8, VCC, VLED outputs
- **Photodiode**: PD5-PD8 inputs with common cathode bias (PDC)

Customization
~~~~~~~~~~~~~

The board features a breakaway line allowing users to separate the two
:adi:`ADPD2140` sensors and manually wire them at custom spacing distances
for tailored gesture recognition applications.

Software and Demo
------------------

A reference gesture sensing demonstration was developed using the
:adi:`EVAL-ADICUP3029` platform. The demo supports four directional gestures
(up, down, left, right) using the dual infrared sensor configuration.
