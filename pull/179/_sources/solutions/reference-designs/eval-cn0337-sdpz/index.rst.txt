EVAL-CN0337-SDPZ
=================

RTD Temperature Sensor Solution.

Overview
--------

:adi:`CN0337` is a complete RTD measurement solution providing temperature
sensing using Pt100 or Pt1000 RTD sensors with direct digital conversion and
integrated signal isolation for robust industrial operation.

The circuit uses the :adi:`AD8608` precision op-amp for signal conditioning and
the :adi:`AD7091R` 12-bit SAR ADC for digitization, connected via an SDP-B
platform or PMOD-compatible interface.

Required Equipment
------------------

- EVAL-CN0337-SDPZ evaluation board
- :adi:`EVAL-SDP-CB1Z` controller board (SDP-B board)
- USB connection to PC
- RTD sensor (Pt100 or Pt1000)

Hardware Setup
--------------

- 12-pin PMOD connector for external microprocessor/FPGA connection.
- 120-pin connector to SDP-B board interface.
- Integrated RTD sensor interface with isolation.

Software Setup
--------------

The evaluation software GUI provides real-time temperature data collection,
chart visualization, and spreadsheet export capabilities.
