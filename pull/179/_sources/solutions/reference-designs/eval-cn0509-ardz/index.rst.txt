EVAL-CN0509-EBZ
================

Wide Input Range (5 V to 100 V) USB Charger.

Overview
--------

:adi:`CN0509` is a wide input range USB charger capable of providing up to 2 A
at 5 V from diverse DC sources such as solar panels, car batteries, telecom
supplies, alkaline cells, wind turbines, and generators. The circuit combines a
wide input :adi:`LTC7103` buck converter with an isolated :adi:`LT8302` flyback
converter.

The input range is 5 V to 100 V. The circuit survives up to 100 V reverse
connection, with polarity indicator LEDs: green (DS1) for correct polarity and
red (DS2) for reversed polarity.

Features
--------

- Wide input range: 5 V to 100 V
- Isolated and regulated 5 V output
- Up to 2 A output current
- Two USB output ports
- Survives 100 V reverse polarity connection
- Polarity indicator LEDs

Required Equipment
------------------

- EVAL-CN0509-EBZ circuit evaluation board
- DC power supply (5 V to 100 V)
- USB multimeter tester (or equivalent)
- USB Type-A to Micro-USB / Type-C / Lightning cable
- USB-capable device (phone, tablet, power bank) with charging cable

Board Layout
------------

- **P1** -- Screw terminal block for input supply
- **P2** -- USB output connectors for load (two ports)
- **DS1** -- Green LED (correct polarity indicator)
- **DS2** -- Red LED (reverse polarity indicator)

Test Points
~~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Test Point
     - Function
   * - TP1
     - :adi:`LTC7103` output voltage programming pin (VPRG2)
   * - TP6
     - Voltage input to :adi:`LTC7103`
   * - PGOOD
     - :adi:`LTC7103` open-drain power good output
   * - CLKOUT
     - :adi:`LTC7103` output clock signal
   * - IMON
     - Average output current monitor (0.4--1.3 V = 0--2.5 A)
   * - ICTRL
     - Output current programming (floating = 2.5 A avg, 3.7 A peak)
   * - 12V
     - 12 V output of :adi:`LTC7103`
   * - GND1
     - Ground relative to input supply
   * - VBUS
     - USB VBUS voltage (~5 V)
   * - D+, D-
     - USB data signals
   * - GND
     - Isolated ground relative to output

Hardware Setup
--------------

#. Connect the input DC supply to **P1** on the EVAL-CN0509-EBZ.

   .. warning::

      Ensure safe connection practices for voltages up to 100 V.

#. Observe the LED indicators:

   - **DS1** (green) = correct polarity, circuit delivers up to 10 W.
   - **DS2** (red) = reverse polarity. Turn off supply, disconnect, swap
     leads, and reconnect.

#. Connect a USB multimeter tester to the lower USB port on **P2**.
#. Connect a fast-charge capable device through the multimeter.
#. Verify the multimeter shows 500 mA to 2 A current draw.

USB Output Ports
~~~~~~~~~~~~~~~~

- **Bottom receptacle** -- USB Dedicated Charging Port (DCP) controller,
  enables up to 2 A charging.
- **Top receptacle** -- General-purpose charging, rated up to 500 mA.

.. note::

   Maximum charging current on the lower port depends on the output voltage to
   the load. The device's charge controller circuitry may limit charging current
   if the load voltage drops below a certain threshold.
