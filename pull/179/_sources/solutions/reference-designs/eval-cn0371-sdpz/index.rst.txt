EVAL-CN0371-SDPZ
=================

Complete LVDT Signal Conditioning Circuit.

Overview
--------

:adi:`CN0371` is a complete linear variable differential transformer (LVDT)
signal conditioning circuit for accurate measurement of linear position or
displacement from a mechanical reference. It uses synchronous demodulation in
the analog domain via the :adi:`ADA2200` for noise immunity and a 24-bit
sigma-delta :adi:`AD7192` ADC for digitization.

The circuit features extremely low power consumption (10 mW total), SPI
programmable excitation frequency and output data rates, and programmable
bandwidth vs. dynamic range trade-off. It supports bandwidths greater than
1 kHz, and at 20 Hz provides 100 dB dynamic range. The contactless,
frictionless operation makes it suitable for aerospace, process controls,
robotics, nuclear, and chemical applications.

Required Equipment
------------------

- :adi:`EVAL-SDP-CB1Z` controller board (SDP-B board)
- :adi:`EVAL-CN0371-SDPZ <CN0371>` evaluation board
- EVAL-CFTL-LVDT (Measurement Specialties E-100 Economy Series LVDT)
- CN0371 Evaluation Software
- +6 V DC power supply (EVAL-CFTL-6V-PWRZ)
- USB Type-A to USB Mini-B cable
- PC with Windows XP SP2 (32-bit), USB port, 1 GHz processor, 512 MB RAM

Hardware Setup
--------------

#. Connect the EVAL-CFTL-LVDT to **J3** (6-pin header) on the
   EVAL-CN0371-SDPZ.

   .. note::

      Wiring differs if using a different LVDT model.

#. Connect **P3** of the CN0371 board to **CON A** of the SDP-B board via the
   120-pin connector.
#. Connect the EVAL-CFTL-6V-PWRZ (+6 V DC power supply) to the CN0371 board
   via the DC barrel jack.
#. Connect the USB cable to **J1** of the SDP-B board.

Software Setup
--------------

Installing the Software
~~~~~~~~~~~~~~~~~~~~~~~~

#. Extract **CN0371 Eval Software.zip** and run **setup.exe**.

   .. note::

      Install to the default path
      ``C:\Program Files\Analog Devices\CN0371\``.

#. Click **Next** through the installation review and start pages.
#. The ADI SDP Drivers installer will execute automatically. Close all other
   applications before proceeding.
#. Set the SDP Drivers installation location (default:
   ``C:\Program Files\Analog Devices\SDP\Drivers``).
#. Complete the installation and click **Finish**.

Using the Evaluation Software
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The evaluation software provides several tabs:

**Linearity and Calibration**

- Displays current displacement in volts and millimeters.
- Two-point calibration: set core at maximum displacement, enter distance,
  click **Set new MAX**; repeat for minimum with **Set new MIN**.
- Linearity table: set core to known positions and click **Get new data point**
  to build a linearity table with automatic error calculation.

**Live Measurement and Noise Data**

- Live chart displays the last 200 ADC readings in volts and millimeters.
- Noise data shows RMS volts, peak-to-peak volts, and ENOBs based on a 3.3 V
  ADC reference.

**Settings**

- **Full-Scale Range** -- Scale voltage for ENOB calculations.
- **Measure I/Q** -- Select Phase 0 or Phase 90 on the :adi:`ADA2200`.
- **Data Rate Divider** -- Set :adi:`AD7192` output data rate (1 to 1023);
  output rate = 4800 / (Data Rate Divider).
